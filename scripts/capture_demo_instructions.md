# Demo Capture Instructions

## Screenshot
1. Run `streamlit run streamlit_app.py`.
2. Pick a scenario that already has media generated.
3. Make sure the page shows the metadata table, preview GIF, RAFT flow, and EPE heatmap.
4. Capture one clean full-window screenshot and save it as `reports/figures/streamlit_screenshot.png`.

## Short Screen Recording
1. Start from the Streamlit home screen with the scenario dropdown visible.
2. Record a 10 to 20 second clip.
3. In the clip, select a scenario, click `Generate media`, and show the GIF plus the two RAFT visualizations.
4. Keep the recording short and stable so it is easy to embed in a presentation.
